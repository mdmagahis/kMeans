///////////////////// ABOUT ///////////////////////////////////
This is a k-means clustering algorithm using the Euclidean 
distance for 2-dimensional numerical data.

For the assignment, this program was written in java using
Eclipse. 


///////////////////// EXECUTUION //////////////////////////////
To execute in Eclipse, adjust 
Run Configurations... > Arguments > Program Arguments 
to the following format:

k inputX.txt

where k is the k-means and X is the int from the input file 
name.

Selecting Run will save output to outputX.txt.
